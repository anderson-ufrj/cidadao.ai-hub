# Contributing to Cidadão.AI Hub

Thank you for your interest in contributing to Cidadão.AI! This document provides guidelines for contributing to the documentation portal.

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or suggest improvements
- Check if the issue already exists before creating a new one
- Provide clear, detailed descriptions

### Documentation Contributions

1. **Fork** the repository
2. **Create a branch** for your changes (`git checkout -b docs/your-improvement`)
3. **Make your changes** following our style guide
4. **Test locally** to ensure documentation renders correctly
5. **Submit a Pull Request** with a clear description

### Style Guide

- Write in clear, accessible language
- Use proper Markdown formatting
- Include code examples where helpful
- Keep paragraphs concise
- Use headings to organize content

### Translation

We welcome translations to make Cidadão.AI accessible to more users:

- Brazilian Portuguese is the primary language
- English translations are appreciated
- Maintain consistent terminology across translations

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold this code.
